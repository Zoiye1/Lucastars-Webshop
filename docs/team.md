## Akif Goge

Ik ben Akif en ik ben 20 jaar. Ik begon met software ontwikkelen op het mbo en ben doorgegaan met deze
opleiding. Sinds ik jong was had ik interesse met het leren hoe software werkten, zoals hoe computerspellen
werken of hoe websites zijn gebouwd. In de toekomst wil ik een fullstack developer worden.

### Semester 1

Het eerste semester begon met blok 1: Dokkie project. Dit is waar we de basistechnieken hadden geleerd met
HTML, CSS en Typescript (in plaats van Javascript). Het ging goed en met mijn mbo voortgang was het makkelijk.

Blok 2 ging wat lastiger want ik moest veel nieuwe technieken leren die ik nooit eerder had gebruikt, zoals je
eigen mvc structuur maken. Plus typescript gebruiken voor de database was iets dat ik nooit eerder had gedaan.
Maar uiteindelijk ging dat ook goed en heb ik samen met teamgenoot alle punten goed gedaan.

### Semester 2

Begin semester 2 waren we begonnen met een spelproject, daar moesten we een spel maken. Ik heb ik veel van
geleerd.

### Project: Webshop

Ik ga dit project vooral proberen eerder en vaker de knowledgebase te gebruiken zodat ik later niet vastloop
over wat wij moeten doen. We moeten ook meer zelfstandig werken zoals userstories maken, dus ik ga
